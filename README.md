# [웹 게임 기획서] 바이오텍 타이쿤: Phase 3 (가제)

## 1. 게임 개요

**장르**: 정통 턴제 경영 시뮬레이션 (Tycoon)

**플랫폼**: 웹 브라우저 (데스크탑 / 모바일 반응형)

**운영 방식**: 오픈소스 프로젝트로 공개 저장소에서 관리

**핵심 컨셉**: 플레이어는 신생 바이오테크 기업의 CEO가 되어, 제한된 자본으로 연구원을 고용하고 신약을 개발하여 회사의 시가총액을 키워야 합니다.

## 4. 핵심 규칙 및 플레이 사이클 (Core Loop)

### 4.1. 핵심 자원

**자본금 (Cash)**: 연구원 월급, 임상 실험 진입 비용 등으로 매 턴 소모됩니다.

**턴 (Turn)**: 1턴 = 1개월. 턴을 넘길 때마다 고정 지출이 발생하고 이벤트가 연산됩니다.

**주가 (Stock Price)**: 회사의 가치이자 자금 조달의 핵심 지표입니다.

**주주 신뢰도 (Trust)**: 너무 잦은 유상증자나 임상 실패 시 하락하며, 낮아지면 주가 방어가 힘들어집니다.

### 4.2. 연구팀 관리 (HR)

각기 다른 스탯(성공률 보정(%), 연구 속도 단축, 월급)을 가진 연구원을 고용합니다.

예산에 맞춰 가성비 좋은 신입을 여러 명 쓸지, 성공 확률을 확실히 높여주는 고액 연봉의 석학을 영입할지 전략적으로 결정해야 합니다.

#### 연구원 등급 예시

| 등급       | 월급     | 성공률 보정 | 속도 단축 | 특징                   |
| ---------- | -------- | ----------- | --------- | ---------------------- |
| **인턴**   | 0.5억/턴 | +1%         | 없음      | 저렴하지만 효과 미미   |
| **주니어** | 1억/턴   | +3%         | 없음      | 가성비 좋은 선택       |
| **시니어** | 3억/턴   | +7%         | -1턴      | 안정적인 핵심 인력     |
| **석학**   | 8억/턴   | +15%        | -2턴      | 고액이지만 확실한 효과 |

#### 연구원 추가 규칙

- **레벨업**: 임상 성공 경험 시 연구원 스탯이 소폭 성장
- **이탈 리스크**: 월급이 시장 평균 이하이면 매 턴 퇴사 확률 발생 (10~30%)
- **팀 시너지**: 같은 질환 분야 전문가 3명 이상 배치 시 성공률 +5% 보너스

### 4.3. 파이프라인 (신약 개발)

#### 타겟 질환 선택

**로우리스크-로우리턴**: 제네릭(복제약) 등은 성공률이 높지만(80%) 주가 상승폭이 적습니다.

**하이리스크-하이리턴**: 혁신 항암제, 희귀질환 치료제 등은 막대한 자금이 들고 성공률이 낮지만(15%), 성공 시 텐배거(10배 주가 폭등)를 달성할 수 있습니다.

#### 임상 단계별 상세 규칙

진행 단계: **전임상 → 임상 1상 → 임상 2상 → 임상 3상 → FDA 승인**

| 단계         | 소요 턴 | 비용  | 기본 성공률 | 성공 시 주가 영향 | 실패 시 주가 영향 |
| ------------ | ------- | ----- | ----------- | ----------------- | ----------------- |
| **전임상**   | 3턴     | 5억   | 70%         | +2%               | -5%               |
| **임상 1상** | 4턴     | 20억  | 60%         | +5%               | -15%              |
| **임상 2상** | 6턴     | 80억  | 40%         | +15%              | -30%              |
| **임상 3상** | 8턴     | 200억 | 30%         | +40%              | -50%              |
| **FDA 승인** | 2턴     | 50억  | 85%         | +100%             | -60%              |

> 위 성공률은 "기본값"이며, 연구원 스탯 보정 및 질환 종류에 따라 변동됩니다.

### 4.4. 임상 결과와 주가 변동 (핵심 재미 요소)

**기대감 반영**: 실제 바이오 주식 시장의 흐름처럼, 임상 진행 중에는 기대감만으로도 주가가 서서히 우상향합니다.

**운명의 룰렛 (턴 종료 시 연산)**: 임상 단계가 끝나는 턴에 확률에 따라 결과가 결정됩니다.

- **성공 시**: 연속 상한가 기록, 다음 단계 진입 가능.
- **실패 시**: 하한가 기록, 해당 파이프라인 즉시 폐기 (투자금 증발), 주주 신뢰도 하락.

### 4.5. 자금 조달 (유상증자)

임상 기대감으로 주가가 고점에 달했을 때 **'유상증자'**를 단행하여 대규모 자본금을 확보할 수 있습니다. 단, 주주 신뢰도가 깎이는 페널티를 감수해야 합니다.

**유상증자 규칙:**

- 조달 금액: 현재 시가총액의 10~30% 범위 내에서 선택
- 신뢰도 페널티: 조달 비율에 비례하여 -5 ~ -20
- 쿨다운: 유상증자 후 6턴 동안 재실행 불가

### 4.6. 특허 & 라이선싱

- **기술 라이선스 판매**: 임상 2상 이후, 대형 제약사에 파이프라인 라이선스를 판매할 수 있음
  - 즉시 현금 확보 (안정적 수입원) 대신, 해당 신약의 향후 주가 상승 효과 50% 감소
  - "지금 안전하게 현금을 받을 것인가 vs 끝까지 가져갈 것인가"의 전략적 선택
- **특허 만료**: FDA 승인 후 24턴(2년) 경과 시 특허 만료 → 제네릭 경쟁 → 해당 신약 수익 80% 감소

### 4.7. 상장폐지 규칙

- 주가가 기준가(난이도별 상이) 이하로 **5턴 연속** 유지 시 → ⚠️ **상장폐지 경고** 발령
- 경고 후 **3턴 내** 기준가 이상으로 회복하지 못하면 → 💀 **게임 오버**
- 경고 기간 중 유상증자는 신뢰도 페널티 2배 적용

## 5. 랜덤 이벤트 시스템

매 턴 종료 시 일정 확률로 이벤트가 발생하여 게임에 변수를 더합니다.

### 5.1. 호재 이벤트

| 이벤트                     | 효과                    |
| -------------------------- | ----------------------- |
| 정부 바이오 육성 정책 발표 | 주가 +10%               |
| 경쟁사 임상 실패 뉴스      | 주가 +5%                |
| 유명 애널리스트 매수 추천  | 주가 +8%, 신뢰도 +5     |
| 국제 학술지 논문 게재      | 진행 중 임상 성공률 +5% |

### 5.2. 악재 이벤트

| 이벤트                     | 효과                      |
| -------------------------- | ------------------------- |
| 임상 데이터 조작 의혹 보도 | 신뢰도 -15                |
| 원자재 가격 폭등           | 진행 중 임상 비용 +20%    |
| 핵심 연구원 스카우트 제의  | 해당 연구원 이탈 확률 50% |
| 공매도 세력 공격           | 주가 -12%                 |

### 5.3. 선택 이벤트

| 이벤트                           | 선택지 A                            | 선택지 B                |
| -------------------------------- | ----------------------------------- | ----------------------- |
| 대형 제약사 파이프라인 매각 제안 | 즉시 현금 확보                      | 거절하고 직접 개발 계속 |
| 정부 보조금 신청 기회            | 보조금 수령 (조건부 성과 보고 의무) | 독립적으로 진행         |
| 경쟁사 합병 제안                 | 합병 수락 (게임 종료, 점수 정산)    | 거절하고 독자 성장      |

## 6. 기술 스택 (Tech Stack)

복잡한 상태 관리와 라우팅, 그리고 화려한 시각화를 완벽하게 지원하기 위한 프론트엔드 중심의 구성입니다.

**코어 프레임워크**: React + NextJS

수많은 UI 컴포넌트를 조립하기 좋고, NextJS의 강력한 라우팅으로 게임 내 여러 화면(오피스, 연구실, 주식 시장)을 매끄럽게 제어합니다.

**전역 상태 관리**: Zustand

턴마다 변하는 자본금, 주가, 연구원 목록, 파이프라인 진행도 등 방대한 게임의 '현재 상태'를 가볍고 직관적으로 관리합니다.

**데이터 시각화**: Recharts (또는 Chart.js)

임상 성공과 실패, 기대감에 따라 요동치는 주가 차트를 실시간으로 그려내어 시각적 타격감을 극대화합니다.

**스타일링 & 애니메이션**: Tailwind CSS + Framer Motion

## 유려한 대시보드 UI를 빠르게 구성하고, 임상 성공 시 팡파르 효과나 실패 시 화면 흔들림 등의 애니메이션을 구현합니다.

## 7. 세이브 & 랭킹 시스템

### 7.1. 클라우드 세이브

- **자동 저장**: 매 턴 종료 시 Firestore에 자동 저장
- **수동 저장**: 플레이어가 원할 때 저장 슬롯 선택 (최대 3개)
- **오프라인 지원**: 비로그인 시 LocalStorage에 임시 저장, 로그인하면 클라우드로 동기화
- **저장 데이터**: 턴 수, 자본금, 주가, 주가 히스토리, 연구원 목록, 파이프라인 진행도, 신뢰도

### 7.2. 랭킹 시스템

| 랭킹 기준            | 설명                               |
| -------------------- | ---------------------------------- |
| **최고 시가총액**    | 게임 종료 시점의 시가총액 기준     |
| **최단 턴 클리어**   | 승리 조건 달성까지 걸린 턴 수      |
| **FDA 승인 신약 수** | 가장 많은 신약을 상용화한 플레이어 |

- **리더보드**: 난이도별로 분리 (이지/노멀/하드 각각 독립 랭킹)
- **시즌제** (선택): 월간 랭킹 초기화로 경쟁 유도
- **게임 오버 시에도 기록**: 파산/상장폐지로 끝나도 "최고 시가총액 도달 기록"은 저장

### 7.3. 인증 (로그인)

- **Google 로그인**: Firebase Auth를 통한 간편 로그인
- **게스트 모드**: 로그인 없이 플레이 가능 (LocalStorage 저장, 랭킹 미등록)
- **게스트 → 회원 전환**: 게스트로 플레이 중 로그인하면 기존 데이터 클라우드로 이전
